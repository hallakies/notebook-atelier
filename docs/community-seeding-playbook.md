# Community Seeding Playbook

Notebook Atelier should enter communities as a helpful buyer, not as a spammy operator.

## Goal

- get the first quiz completions
- learn which buyer questions repeat
- convert high-intent threads into site visits

## Where to participate

- Apple / MacBook question threads
- laptop comparison threads
- student purchase advice threads
- developer setup threads

## Rules

- never drop a naked link first
- answer the question in 3 to 5 lines before linking
- do not claim authority you do not have
- do not pretend to be Apple or an official seller
- do not automate posting or comments

## Reply structure

1. restate the user problem briefly
2. give a direct answer
3. explain why in one short paragraph
4. if useful, offer the quiz link as a faster shortcut

## Daily loop

1. find 3 relevant threads
2. answer 1 or 2 with real value
3. log which question theme repeated
4. turn repeated themes into new guide topics
