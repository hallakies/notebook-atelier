# First 100 Users Playbook

Notebook Atelier should not chase broad awareness first. The first 100 users should come from people who are already close to buying a MacBook.

## Position

- Product: conversion-first MacBook buying tool
- Market: Korean buyers
- Core action: quiz completion -> recommendation -> Coupang click

## First channels

### 1. Search

Primary target keywords:

- 맥북 추천
- 맥북 에어 vs 프로
- 학생 맥북 추천
- 개발자 맥북 추천
- 직장인 맥북 추천
- 맥북 지금 사도 될까

Landing mapping:

- home -> 맥북 추천
- /buying-guides/macbook-air-vs-pro -> 맥북 에어 vs 프로
- /buying-guides/best-macbook-for-students -> 학생 맥북 추천
- /buying-guides/best-macbook-for-developers -> 개발자 맥북 추천
- /buying-guides/best-macbook-for-office-work -> 직장인 맥북 추천
- /buying-guides/should-i-buy-a-macbook-now -> 맥북 지금 사도 될까

### 2. Naver Blog

Treat Naver Blog as the distribution layer for search, not as the source of truth.

Workflow:

1. Publish or update a buying guide on the site
2. Generate a Slack brief
3. Rewrite lightly for a human Naver Blog tone
4. End with a CTA back to the quiz

### 3. Instagram

Treat Instagram as recall and trust, not the first conversion layer.

Workflow:

1. Reuse the guide topic
2. Create one carousel
3. Use the image prompt from Slack
4. Link back to the site quiz

## Daily operating loop

1. Check Search Console queries
2. Pick 1 keyword that already has intent
3. Improve the matching landing page
4. Send 1 Naver Blog brief to Slack
5. Send 1 Instagram brief to Slack
6. Publish both manually

## First experiments

### Experiment A

- Topic: 맥북 추천
- Goal: get the first quiz completions from search and blog

### Experiment B

- Topic: 맥북 에어 vs 프로
- Goal: get the first comparison-intent visitors

### Experiment C

- Topic: 학생 맥북 추천
- Goal: capture a narrow but clear buyer segment

## Success signals

- Search impressions begin to appear
- Quiz completions are increasing
- Product block clicks happen without direct prompting
- One topic clearly outperforms the others

## Anti-patterns

- Publishing many weak topics at once
- Treating Instagram as the main sales channel too early
- Writing AI-sounding copy with weak buying intent
- Chasing general Apple news instead of buyer questions
